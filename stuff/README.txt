LOXYS DEN - Website Files
=========================

Copy ALL files from this folder into your web hosting file manager.

Files included:
- index.html      (Home)
- mines.html      (Mines game)
- crash.html      (Crash game)
- login.html      (Login/Register)
- admin.html      (Admin panel)
- leaderboard.html (Leaderboard)
- auth-api.js     (Auth & API)
- config.js       (API URL - edit this for your backend!)

IMPORTANT: Edit config.js
- For local: Leave LOXYS_API empty (uses localhost:3000)
- For hosted: Set window.LOXYS_API = 'https://your-backend-url.com';

The backend must run separately (Node.js server).
See the main loxys-den folder for backend files.
